Welcome to the Mozilla Plugin Directory

This site is an attempt to collect and provide information about third-party browser plugins installed by people across the web.

Using our plugin detection service, you can check to see if your plugins are up to date (as far as we know) or if you may be vulnerable to any problems or security issues.

We can also use your help: If you are a plugin vendor, or simply know something we don't know about a plugin, you can help us by submitting suggested updates to this database along with information we've detected about your browser and installed plugins.

If you want to get even more involved, you can sign up for a profile. This will give you access to a sandbox in which to edit plugin records, test them against our detection code in browsers, and submit changes to our editors for approval and inclusion in the database.
